## JavaScript function


### funtion naming rule  

함수는 어떤 **동작**을 수행하기 위한 코드를 모아 놓은 것.  
어떤 동작인지 축학해서 동사를 접두어로 붙여 함수 이름을 만드는 게 관습이다.  
  
| 접두어       | 의미                               | 예              |  
|------------- |:---------------------------------:|----------------:|
| `show~`      | 무언가 보여줌                      |shwMessage(..)   |
| `get~`       | 값을 반환                          | getAge(..)      |
| `calc~`      | 무언가 계산                        | calcSum(..)     |
| `create~`    | 무언가 생성                        | createForm(..)  |
| `check~`     | 무언가 확인하고 boolean값을 반환    | checkPermission(..) |
